# Pairwise Sequence Alignment Analysis

## 📌 Project Overview
This project focuses on performing pairwise sequence alignment using bioinformatics tools and analyzing sequence similarity.

## 🧬 Objectives
- Retrieve gene sequence from NCBI (KRAS gene)
- Perform BLAST analysis
- Interpret alignment results
- Compare global and local alignment methods

## ⚙️ Tools & Technologies
- NCBI GenBank
- BLAST (NCBI)
- EMBOSS (Needle, Water)
- FASTA format

## 🧪 Methodology
1. Retrieved KRAS gene sequence from NCBI
2. Performed BLASTN search against database
3. Analyzed alignment score, E-value, and identity
4. Performed:
   - Global alignment (Needleman-Wunsch)
   - Local alignment (Smith-Waterman)

## 📊 Results
- Identified highly similar sequences (up to ~99% identity)
- Observed strong alignment scores and low E-values
- Compared global vs local alignment outputs

## 🚀 Key Learnings
- Understanding of sequence similarity and alignment
- Practical use of BLAST and EMBOSS tools
- Interpretation of biological sequence data
